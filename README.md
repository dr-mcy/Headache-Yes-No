# Headache-Yes-No

頭痛があるとき・頭痛がないときチェックシート（HIT-6™ + MIBS-4）の単一HTML WEBアプリ。

## 特徴

- **日本語版HIT-6™**（6問×5択、合計36〜78点）と **日本語版MIBS-4**（4問×6択、合計0〜12点）を1ページに統合。
- 選択肢タップで自動採点し、評価シートに沿った支障度判定を色分けで表示。
- 紙の問診票（イーライリリー・第一三共 PP-GZ-JP-2646）のレイアウトを踏襲。
- iPad / PC 両対応、印刷レイアウト対応。
- 単一HTML・依存ライブラリなし（オフライン動作）。

## 使い方

1. リポジトリをクローン or `index.html` をダウンロード。
2. ブラウザで `index.html` を開く（ローカルファイルでOK）。
3. 患者情報を記入し、設問の選択肢をタップ。
4. 全問回答後、HIT-6 判定 / MIBS-4 判定が自動表示される。
5. 「印刷」でカルテ貼付用にA4出力、「リセット」で初期化。

## GitHub Pages で公開する

リポジトリの **Settings → Pages** で `main` ブランチの `/ (root)` を公開元に設定すると、`https://dr-mcy.github.io/Headache-Yes-No/` で動作します。

## 配点ロジック

### HIT-6（頭痛があるとき）

| 列 | ラベル例 | 配点 |
|---|---|---|
| 1 | 全くない / 全くなかった | ×6 |
| 2 | ほとんどない / ほとんどなかった | ×8 |
| 3 | 時々ある / 時々あった | ×10 |
| 4 | しばしばある / しばしばあった | ×11 |
| 5 | いつもそうだ / いつもそうだった | ×13 |

**判定**: 60点以上=重度の支障 / 56-59=かなりの支障 / 50-55=ある程度の支障 / 49点以下=影響なし

### MIBS-4（頭痛がないとき）

| 列 | ラベル | 配点 |
|---|---|---|
| 1 | わからない／該当なし | ×0 |
| 2 | 全くなかった | ×0 |
| 3 | ほとんどなかった | ×1 |
| 4 | 時々 | ×2 |
| 5 | 多くの時間 | ×3 |
| 6 | ほぼいつも／いつも | ×3 |

**判定**: 5点以上=重度 / 3-4=中等度 / 1-2=軽度 / 0=支障なし
